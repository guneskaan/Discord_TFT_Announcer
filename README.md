## Running the Bot
1) Obtain bot token from Discord Developer Portal
2) Create `.env` file in the main repository in the following format:
    ```
    DISCORD_TOKEN=<YOUR DEVELOPER TOKEN>
    ```
3) Start bot with
    ```
    node bot.js
    ```
